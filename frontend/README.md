# 🚀 Modern React Portfolio | M. Sajjad

Welcome to my professional portfolio! This project is a highly interactive, modern, and responsive web application built using **React.js**, **Framer Motion**, and Tailwind CSS. It features a sleek terminal-inspired aesthetic and smooth transitions.

## 🌐 Live Link
**View my portfolio here:** [https://msajjad-swart.vercel.app/](https://msajjad-swart.vercel.app/)

## ✨ Key Features
- **Modern UI/UX**: Dark-themed, professional layout with a focus on typography.
- **Responsive Navigation**: Fully functional sidebar for mobile and desktop.
- **Optimized Build**: Configured to bypass common linting warnings for seamless Vercel deployment.
- **Dynamic Content**: Sections for Home, About, Skills, Projects, and Contact.
- **Interactive Animations**: Smooth scroll and hover effects using Framer Motion.

## 🛠️ Tech Stack
- **Frontend**: React.js (v19)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: React Icons
- **Deployment**: Vercel

## 🔧 Deployment Fixes
During development, several critical build issues were resolved:
- **Build Loop Fix**: Corrected the `npm run build` infinite loop in `package.json`.
- **Linting Bypass**: Implemented `CI=false` to handle unused variables and security warnings during production.
- **Dependency Management**: Updated all necessary libraries including `react-icons` for deployment.
